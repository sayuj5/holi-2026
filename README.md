# 🌈 Happy Holi 2026 - Festive Wish Website

A vibrant, hyper-realistic, and interactive Holi-wishing website created for the 2026 celebrations. This project features dual-language support (English & Bengali) with beautiful, animated song messages.

## ✨ Features

- **Premium UI**: Designed with glassmorphism, glowing text, and festive animations.
- **Dual Language**: Support for English and Bengali wishes with tailored greetings.
- **Staggered Song Messages**: Beautifully animated display of iconic Holi song lyrics:
  - **English**: First 4 lines of "Balam Pichkari".
  - **Bengali**: First 4 lines of "Rangiye Diye Jao" (Iman Chakraborty).
- **High-Resolution Visuals**: Features a stunning, AI-generated Holi background with blur transitions.
- **Responsive Design**: Optimized for a seamless experience on mobile, tablet, and desktop.

## 🚀 Live Demo

Experience the festive magic here: [https://sayuj5.github.io/holi-2026/](https://sayuj5.github.io/holi-2026/)

## 🛠️ Built With

- **HTML5**: Semantic and clean structure.
- **Vanilla CSS3**: Custom animations, glowing effects, and a glassmorphism design system.
- **JavaScript (ES6+)**: Core logic for language switching and staggered message rendering.

## 📁 Project Structure

- `index.html`: Main entry point and structural layout.
- `style.css`: Comprehensive design tokens and animation logic.
- `script.js`: Dynamic content management and interactive features.
- `bg.png`: vibrant festive background asset.
